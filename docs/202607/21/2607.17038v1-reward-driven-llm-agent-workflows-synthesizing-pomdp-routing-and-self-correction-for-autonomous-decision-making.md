## 摘要
本文通过设计和优化智能智能体工作流，解决了当前大语言模型（LLM）智能体应用中的关键技术挑战，包括长时域规划、稀疏奖励归因以及动态环境交互。提出的架构基于核心人工智能范式的综合：视觉、语言、生成式、图、多模态、强化学习和智能体智能。与依赖静态提示且缺乏稳健感知-行动循环的传统基线模型不同，我们的方法引入了一种部分可观测马尔可夫决策过程（POMDP）路由机制。该机制通过一个内在的、自我修正的奖励模型增强，该模型在执行前评估决策轨迹。通过整合多模态输入和先进的强化学习原理（如近端策略优化和价值函数近似），智能体保持长期结构记忆，并动态调整其推理路径以减少误差累积。在ALFWorld具身模拟环境和WebShop在线导航基准上的实验表明，与标准ReAct框架等主流基线相比，任务成功率和轨迹效率绝对提升了24.5%。全面的消融研究证实了奖励驱动批评模块在抑制幻觉率方面的重要贡献。本研究将强化学习和基于图的记忆的理论基础与自主智能体工作流相连接。最终，所提出的架构为在复杂的多步骤自主系统中开发人工智能技术提供了一个实用、可扩展的参考框架。代码可在https://github.com/01Amez/RLAW_Implementation获取。

## Abstract
This paper addresses key technical challenges in current large language model (LLM) agent applications, including long-horizon planning, sparse reward attribution, and dynamic environmental interaction, by designing and optimizing an intelligent agent workflow. The proposed architecture is based on the synthesis of core AI paradigms: Visual, Language, Generative, Graph, Multimodal, Reinforcement, and Agent Intelligence. Unlike conventional baseline models that rely on static prompting and lack robust perception-action loops, our approach introduces a Partially Observable Markov Decision Process (POMDP) routing mechanism. This mechanism is augmented with an internal, self-correcting reward model that evaluates decision trajectories before execution. By integrating multimodal inputs and advanced reinforcement learning principles (such as proximal policy optimization and value function approximation), the agent maintains long-term structural memory and dynamically adapts its reasoning pathways to mitigate error accumulation. Empirical experiments on the ALFWorld embodied simulation environment and the WebShop online navigation benchmark demonstrate a 24.5% absolute improvement in task success rate and trajectory efficiency over mainstream baselines like the standard ReAct framework. Comprehensive ablation studies confirm the significant contribution of the reward-driven critique module in suppressing hallucination rates. This research bridges theoretical foundations of reinforcement learning and graph-based memory with autonomous agent workflows. Ultimately, the resulting architecture offers a practical, scalable reference framework for developing artificial intelligence technologies in complex, multi-step autonomous systems. Code is available at https://github.com/01Amez/RLAW_Implementation.

### 1. 核心问题与整体含义（研究动机和背景）

当前大语言模型（LLM）驱动的智能体在处理复杂、多步骤的自主决策任务时面临三大挑战：**长时域规划易出现错误累积**（早期推理错误导致后续步骤全线崩溃）、**稀疏奖励归因困难**（环境仅在最终步骤提供反馈，中间动作缺乏有效引导）、以及**动态部分可观测环境下的交互鲁棒性不足**（无法感知完整状态，容易陷入死循环或执行无效动作）。现有主流方法如ReAct虽然具备“推理-行动”循环，但本质上仍依赖最大似然生成，缺乏对动作长期后果的形式化评估，导致“级联幻觉”频发。本文旨在通过将LLM智能体问题形式化为部分可观测马尔可夫决策过程（POMDP），并引入内在的、奖励驱动的自我修正机制，从根本上提升智能体在动态环境中的决策质量和可靠性。

### 2. 论文提出的方法论

- **核心思想**：提出**RLAW（Reward-Driven LLM Agent Workflow）**，将智能体循环分解为“生成（Propose）-批评（Critique）-执行（Execute）”三个阶段，其中批评阶段由一个轻量级内部奖励模型（Critic）对候选动作进行评估，只有通过阈值（τ=0.7）的动作才被实际执行，否则要求Actor根据反馈重新生成（最多3次重试）。
- **关键技术细节**：
  - **POMDP路由**：将环境建模为7元组⟨S,A,T,R,Ω,O,γ⟩，智能体维护一个**图记忆模块M**（使用图注意力网络GAT更新），作为对不可观测信念状态b_t的近似。
  - **Actor（LLM）**：基于LLaMA-3-8B，根据目标、历史上下文和序列化的图记忆，自回归生成推理轨迹z_t和候选动作a_t。
  - **Critic（内部奖励模型）**：一个经过LoRA微调的轻量模型f_ϕ，接收目标、历史与候选动作，输出一个JSON格式的评估结果（逻辑检查、安全检查和批评分数），并使用交叉熵损失训练。内部奖励R_critique = σ(f_ϕ(…))。
  - **强化学习优化**：总回报J(π_θ) = E[∑γ^t (R_env + λ R_critique)]，使用PPO中的截断代理目标函数更新策略，价值函数V_ϕ通过贝尔曼方程估算。
  - **图记忆更新**：每步通过f_update(M_{t-1}, o_t, a_{t-1})更新节点与边，使用GAT注意力系数α_{i,j}进行邻居聚合。
- **算法流程**：每步t → 生成候选动作a_t → Critic计算R_critique → 若R_critique < τ则拒绝并反馈，Actor重新生成 → 循环至多3次 → 执行动作，获得环境奖励R_env和新观测o_{t+1}。

### 6. 论文的主要结论与发现

- RLAW在ALFWorld上取得78.6%成功率（比ReAct 54.1%绝对提高24.5%），WebShop上65.8%（比42.3%提高23.5%）。

- 图记忆模块避免了“忘记先前状态”导致的回溯，进一步提升效率。

### 7. 优点

- **方法创新**：将POMDP形式化、内在奖励模型与LLM生成深度融合，提出了“生成-批评-执行”的闭环，提供了数学上严格的自我修正机制。

- **实用性强**：使用开源模型和4-bit量化，单卡可运行，代码开源，便于复现和工业部署。
- **安全性优先**：批评机制在低延迟代价下提供了高可靠性，防止错误传播，适用于金融、机器人等高危场景。

### 8. 不足与局限

- **推理延迟**：每步2.15秒（约为ReAct的1.8倍），不适合实时要求高的系统（如高速无人机）。

- **硬件依赖**：虽然单卡可跑，但Critic训练需要多GPU，且推理仍需要24GB VRAM，对资源有限场景不够友好。
