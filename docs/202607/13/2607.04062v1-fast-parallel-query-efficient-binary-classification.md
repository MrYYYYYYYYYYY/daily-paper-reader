## 摘要
我们研究基本的分类问题：为大小为$n$的二元标记数据集计算分离超平面，该数据集具有归一化的$d$维特征。令$\Phi\in \mathbb{R}^{n \times d}$表示特征矩阵，$γ$表示最大间隔分离超平面的间隔，我们提出一种随机算法，该算法在$\tilde{O}(γ^{-2/3}\, \operatorname{nnz}(\Phi) + γ^{-2(ω+1)/3})$的序列运行时间（工作量）、$\tilde{O}(γ^{-2/3})$的并行（计算）深度内解决该问题，并且仅通过$\tilde{O}(γ^{-2/3})$次矩阵-向量查询（matvecs）访问$\Phi$。我们还提出了第二种更快的随机算法，其序列运行时间为$\tilde{O}(γ^{-2/3}\, \operatorname{nnz}(\Phi) + γ^{-2})$，使用$\tilde{O}(γ^{-2/3})$次对$\Phi$的矩阵-向量查询，但仅达到$\tilde{O}(γ^{-4/3})$的并行深度。两种算法均匹配了Kornowski和Shamir [2025]、Karmarkar等人 [2026] 最近建立的近乎最优的确定性矩阵-向量复杂度，并在牺牲随机性的代价下实现了改进的序列运行时间和并行深度。

## Abstract
We study the fundamental classification problem of computing a separating hyperplane for a binary-labeled dataset of size $n$ with normalized $d$-dimensional features. Letting $Φ\in \mathbb{R}^{n \times d}$ denote the feature matrix and $γ$ the margin of the maximum-margin separating hyperplane, we present a randomized algorithm that solves this problem in $\tilde{O}(γ^{-2/3}\, \operatorname{nnz}(Φ) + γ^{-2(ω+1)/3})$-sequential running time (work), $\tilde{O}(γ^{-2/3})$-parallel (computational) depth, and accesses $Φ$ only through $\tilde{O}(γ^{-2/3})$-matrix-vector queries (matvecs). We also present a second, faster randomized algorithm with a $\tilde{O}(γ^{-2/3}\, \operatorname{nnz}(Φ) + γ^{-2})$-sequential running time that uses $\tilde{O}(γ^{-2/3})$-matvecs to $Φ$, but achieves only $\tilde{O}(γ^{-4/3})$-parallel depth. Both algorithms match the near-optimal deterministic matvec complexity recently established by Kornowski and Shamir [2025], Karmarkar et al. [2026] and achieve improved sequential runtime and parallel depth, albeit at the expense of using randomness.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思路
- **步骤1：转化为 ℓ₂-ℓ₁ 矩阵博弈**  
  通过标准归约（Lemma 3.2），将分离超平面问题等价为求解一个 $\rho$-游戏的 $\epsilon$-解，其中游戏矩阵为 $A = -\Phi_D$，且满足 $\|A\|_{2\to\infty} \le 1$。

- **步骤2：球加速框架（Ball Acceleration）**  
  利用平滑技巧（softmax 函数）将非光滑目标 $f(x)=\max_{y\in\Delta_n} y^\top A x$ 替换为光滑代理 $f_{\epsilon'}(x)$（$\epsilon' = \epsilon/(2\log n)$）。然后使用**球加速**（Proposition 4.16）将优化问题分解为 $\tilde{O}(\epsilon^{-2/3})$ 个**正则化子问题**：
  \[
  \min_{x\in B_r(\bar x)\cap B_d} g_{\nu,\bar x}(x)=f_{\epsilon'}(x)+\frac{\nu}{2}\|x-\bar x\|_2^2,
  \]
  其中 $\nu=\tilde{\Omega}(\epsilon^{-1/3})$。

- **步骤3：转化为线性系统求解**  
  利用 Hessian 稳定性（由于 $f_{\epsilon'}$ 是准自协和函数），每个正则化子问题可进一步通过**牛顿法**（Lemma 4.11）转化为 $\tilde{O}(1)$ 个**双球约束二次规划问题**（trust-region with two balls）。通过二维对偶空间的切平面法，每个二次规划又归结为求解高精度的线性系统：
  \[
  (H + \nu I) y = \hat g,\quad H = \frac{1}{\epsilon'} A^\top (P_{\bar x} - p_{\bar x} p_{\bar x}^\top) A.
  \]

- **步骤4：高效线性系统求解**  
  - **化简**：利用 Sherman-Morrison 公式将 $H$ 转化为 $B^\top B + \nu I$，其中 $B = \frac{1}{\sqrt{\epsilon'}} P_{\bar x}^{1/2} A$。
  - **随机子空间嵌入**：采用两个稀疏 oblivious 子空间嵌入矩阵 $S\in\mathbb{R}^{s\times n}$ 和 $T^\top\in\mathbb{R}^{s\times d}$（$s = \tilde{O}(\epsilon^{-2/3})$），使得 $B^\top B + \nu I \approx_2 B^\top S^\top S B + \nu I$，以及 $S B B^\top S^\top + \nu I_s \approx_2 S B T T^\top B^\top S^\top + \nu I_s$。
  - **预条件化 Richardson 迭代**：通过 Woodbury 恒等式将 $d\times d$ 系统转化为 $s\times s$ 系统，并使用预条件 Richardson 迭代求解。
  - **两种底层求解器**：
    - **FMM（快速矩阵乘法）**：直接对 $s\times s$ 矩阵求逆，工作量为 $\tilde{O}(s^\omega)$，深度 $\tilde{O}(1)$。
    - **SVRG（随机方差缩减梯度下降）**：求解最小二乘问题，工作量为 $\tilde{O}(\text{nnz}(B) + \|B\|_F^2/\nu) = \tilde{O}(\text{nnz}(A) + s^2)$，深度 $\tilde{O}(\|B\|_F^2/\nu) = \tilde{O}(\epsilon^{-4/3})$。

- **步骤5：样本重用（Sample Reuse）**  
  由于 $T$ 是随机采样的且与迭代点 $\bar x$ 无关，应用 [Jin 等 2026] 的框架，可**重复使用**同一个 $T$ 在所有外循环迭代中，从而避免每次重新构造 $AT$，将每轮线性系统求解的额外工作量降低至 $\tilde{O}(\text{nnz}(A))$。
