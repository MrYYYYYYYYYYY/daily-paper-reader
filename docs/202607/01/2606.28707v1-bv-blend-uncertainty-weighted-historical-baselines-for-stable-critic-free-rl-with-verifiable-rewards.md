## 摘要
无评论器的可验证奖励强化学习（RLVR），以组相对策略优化（GRPO）为例，避免了训练价值函数（评论器），相较于基于评论器的PPO流程在大型语言模型对齐中减少了内存和计算开销。然而，GRPO风格的优势估计依赖于提示本地（组内）奖励统计，可能不稳定。特别是当提示组中的所有 rollout 获得相同奖励时，组内奖励方差为零，组归一化导致该组优势为零，阻碍了使用二元验证器的冷启动机制中的学习。我们引入了BV-Blend，一种无评论器框架，通过结合提示本地的在策略统计与语义聚类条件的历史矩来稳定优势估计。BV-Blend为每个聚类维护EMA追踪的奖励矩，从均值标准误差（SEM）代理中推导出置信权重，并使用该权重将历史与提示本地基线和方差统计混合，形成用于PPO风格裁剪更新的标准化优势。在可验证推理基准上的实验表明，BV-Blend提高了训练稳定性和性能，并在组归一化方法可能停滞的情况下保持鲁棒性。

## Abstract
Critic-free reinforcement learning with verifiable rewards (RLVR), exemplified by Group Relative Policy Optimization (GRPO), avoids training a value function (critic) and reduces memory and compute overhead relative to critic-based PPO pipelines for aligning large language models. However, GRPO-style advantage estimation depends on prompt-local (within-prompt-group) reward statistics and can be unstable. In particular, when all rollouts in a prompt group receive identical rewards, the within-group reward variance becomes zero, and group normalization yields zero advantages for that group, impeding learning in cold-start regimes with binary verifiers. We introduce BV-Blend, a critic-free framework that stabilizes advantage estimation by combining prompt-local on-policy statistics with semantic-cluster-conditioned historical moments. BV-Blend maintains EMA-tracked reward moments for each cluster, derives a confidence weight from a standard error of the mean (SEM) proxy, and uses this weight to blend historical and prompt-local baseline and variance statistics into a standardized advantage for PPO-style clipped updates. Experiments on verifiable reasoning benchmarks show that BV-Blend improves training stability and performance, and remains robust in regimes where group-normalized methods may stall.

## 1. 论文的核心问题与整体含义

- **研究动机**：在可验证奖励强化学习（RLVR）中，无评论器方法（如GRPO）通过提示组内奖励归一化计算优势，优点在于避免训练价值函数，节省内存和计算。但该方法在“冷启动”阶段或二元验证器场景下存在严重不稳定性：当某个提示组的所有生成轨迹获得相同奖励（如全部错误或全部正确）时，组内奖励方差为零，导致归一化后所有优势变为零，学习信号完全消失，从而使该组的训练停止。
- **整体目标**：提出一种无需评论器的优势估计改进框架，在保持PPO风格裁剪目标不变的前提下，通过融合历史统计信息来稳定优势估计，特别解决零方差组的学习信号崩溃问题。

## 2. 论文提出的方法论

- **核心思想**：为每个提示维护语义聚类，并对每个聚类采用指数移动平均（EMA）追踪历史奖励均值和二阶矩；根据历史估计的可靠性（通过均值标准误差SEM代理）计算置信权重，将历史统计与提示组内统计进行混合，得到最终的基线（baseline）和尺度（scale），再进行一次标准化得到优势值。
- **关键技术细节**：
    - **语义聚类**：使用冻结的句子编码器将提示嵌入，并利用离线训练的K-means码本将提示分配到固定聚类（K个）。聚类码本在RL训练中保持不变。
    - **历史统计维护**：对每个聚类k维护EMA均值 m₁(k)、EMA二阶矩 m₂(k) 和有效样本数 N_eff(k)。更新使用率 γ=0.9，冷启动初始化时设置 N₀ 和 V_prior。
    - **置信权重**：定义历史不确定性代理 SEM_hist(k) = σ_hist(k)/√(N_eff(k)+δ_N)，其中 σ_hist 为历史标准差。置信权重 w_k = exp(-SEM_hist(k)/T)，T为温度超参数。w_k∈(0,1]，历史不确定性越低，w_k越大。
    - **混合与标准化优势**：
        - 混合基线 b(m) = w·μ_hist + (1-w)·μ_local
        - 混合方差 s²(m) = w·v_hist + (1-w)·(σ_local)²
        - 标准化优势 A_BV = (R - b) / (s + δ)
    - **更新顺序**：置信权重基于更新前的EMA计算，EMA使用当前批数据在策略更新后更新。对于首次出现的聚类，w设为0，退化为纯组内归一化。
- **算法流程**：每个迭代，从当前策略采样M个提示、每个提示G条轨迹 → 计算奖励 → 根据聚类和EMA计算BV-Blend优势 → 使用PPO裁剪目标更新策略 → 聚合每个聚类的充分统计量，更新EMA缓冲区。
