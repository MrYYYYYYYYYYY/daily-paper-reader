## 摘要
大型视觉语言模型（LVLMs）在图像描述和视觉问答等视觉理解任务中取得了显著进展。然而，它们仍然容易产生幻觉，生成与实际视觉输入不一致的内容。现有方法主要干预解码阶段，同时忽视了幻觉的关键来源：误导解码过程的不相关或噪声视觉令牌。为了解决这个问题，我们提出了SeeMe，一种无需训练的框架，将传统机器学习中的特征工程概念引入LVLMs。SeeMe通过三阶段令牌工程过程重构视觉令牌，以抑制幻觉来源，同时保留信息丰富的视觉证据。在四个LVLMs上的MME、POPE和AMBER基准测试中，SeeMe持续减少幻觉并提高输出一致性，为缓解LVLMs中的幻觉提供了新视角。

## Abstract
Large Vision-Language Models (LVLMs) have achieved remarkable progress in visual understanding tasks such as image captioning and visual question answering. However, they remain susceptible to hallucinations, generating content that is inconsistent with the actual visual input. Existing methods primarily intervene at the decoding stage, while overlooking a critical source of hallucinations: irrelevant or noisy visual tokens that mislead the decoding process. To address this issue, we propose SeeMe, a training-free framework that introduces the concept of feature engineering from traditional machine learning into LVLMs. SeeMe restructures visual tokens through a three-stage token engineering process to suppress hallucination sources while preserving informative visual evidence. Experiments on MME, POPE, and AMBER benchmarks across four LVLMs demonstrate that SeeMe consistently reduces hallucinations and improves output consistency, providing a novel perspective for mitigating hallucinations in LVLMs.

## 2. 方法论：核心思想、关键技术细节
**核心思想**：通过三阶段“令牌工程”（Token Engineering）逐步重构视觉令牌序列：先粗剪（Selection）、再语义融合扩充（Merging）、后精选（Selection）。整个过程无需训练，仅依赖模型已有的交叉注意力和自注意力机制。

### 三阶段流程（伪代码见于算法1）：
1. **阶段1：跨模态注意力选择器（Cross-Modal Attention Selector）**
   - 在特定解码器层（如第K层）提取交叉注意力权重，计算每个视觉令牌对所有文本令牌（查询+已生成响应）的平均注意力得分 `r_i`。
   - 按得分排序，保留前 `k = ⌊P·Lv⌋` 个令牌（P为保留比例，通常设为0.05）。硬性剪除低分令牌，减少噪声。
2. **阶段2：高质量令牌扩展器（High Quality Token Expander）**
   - 动机：阶段1的剪枝可能误删包含局部重要细节的令牌（如边缘、小物体），导致信息丢失。
   - 方法：基于原始嵌入层的视觉令牌，计算余弦相似度矩阵（屏蔽自相似），为每个令牌找到最相似的t个邻居，进行加权融合：`ˆx_i^(j) = α_ij·x_i + (1-α_ij)·x_j`，其中α_ij = s_ij/(s_ij+1)。
   - 融合产生的k*t个新令牌与原始令牌拼接，形成丰富后的令牌池。
3. **阶段3：跨模态令牌精炼器（Cross-Modal Token Refiner）**
   - 对阶段2生成的融合令牌，再次利用交叉注意力计算每个融合令牌从文本接收的平均注意力分数。
   - 仅保留分数最高的n个融合令牌（如54个），与阶段1保留的剪枝令牌拼接，作为最终视觉表示输入后续解码。

- 效率方面，SeeMe以最低的预填充延迟和最高吞吐量实现了最佳幻觉抑制，实际部署可行。
- 关键洞察：在中间层（约40-50%层数）进行剪枝效果最好，保留比例5%就足够；过早或过晚剪枝效果不佳。
