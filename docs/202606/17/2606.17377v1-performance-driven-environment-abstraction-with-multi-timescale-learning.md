## 摘要
我们研究了大马尔可夫决策过程中决策的性能驱动环境抽象。与保留几何或拓扑结构不同，我们寻求直接优化决策质量的抽象。我们将抽象建模为通过聚合状态空间并在每个聚合状态内强制执行共享动作分布而获得的受控近似。对于固定划分，我们建立了性能保证，将值函数逼近误差与动作共享引入的损失分开。在此分析的指导下，我们开发了一个多时间尺度强化学习框架，该框架联合调整策略和树状结构的环境抽象。由此产生的算法基于Q值差异细化和粗化状态空间的区域，平衡性能与抽象大小和复杂性。实验结果表明，与演员-评论家基线相比，实现了显著的状态压缩、改进的样本效率和更快的重新规划能力。

## Abstract
We study performance-driven environment abstraction for decision-making in large Markov decision processes. Rather than preserving geometric or topological structure, we seek abstractions that directly optimize decision quality. We model abstraction as a controlled approximation obtained by aggregating the state space and enforcing a shared action distribution within each aggregated state. For a fixed partition, we establish a performance guarantee that separates value-function approximation error from the loss introduced by action sharing. Guided by this analysis, we develop a multi-timescale reinforcement learning framework that jointly adapts the policy and a tree-structured environment abstraction. The resulting algorithm refines and coarsens regions of the state space based on Q-value discrepancies, balancing performance against abstraction size and complexity. Empirical results demonstrate substantial state compression, improved sample efficiency, and faster replanning compared to actor-critic baselines.

- **背景**：在大规模马尔可夫决策过程（MDP）中，直接对原始状态空间进行规划和决策在计算上不可行，尤其在安全关键场景中需要快速反应。人类和智能体通过形成环境的分层抽象来管理复杂性，但现有抽象方法大多保持几何或拓扑结构，或使用信息论、结构启发式方法，未能直接优化决策质量。
- **核心问题**：如何自动构建**性能驱动**的状态空间抽象，即抽象不是基于几何或拓扑相似性，而是直接优化下游决策性能，同时平衡抽象复杂度和决策质量。
- **含义**：提出一种框架，能够在线联合学习策略和树状状态聚合，使得抽象动态细化和粗化，仅当性能有提升时才增加分辨率，从而实现大幅状态压缩、样本效率提升和跨任务快速重规划。

## 2. 方法论：核心思想、关键技术细节

### 核心思想

- 将环境抽象建模为**受控的近似**：通过聚合状态空间（形成超状态），并强制每个超状态内所有状态共享相同的动作分布（Same-Action-Distribution, SAD 约束）。决策在抽象层（超状态空间）上进行，减少策略搜索空间。
- 推导了固定聚合下的**性能保证**，将性能损失分解为两部分：
  - **值函数近似误差** \(\epsilon_\Gamma\)：用分段常数函数逼近最优值函数带来的误差。
  - **动作共享损失** \(\delta_\Gamma\)：强制超状态内所有状态共享同一动作分布导致的额外损失。
- 基于此分析，设计了**多时间尺度强化学习框架**，在三个时间尺度上分别更新：
  1.  **最快时间尺度**：超状态内的 critic（值函数）更新。
  2.  **中间时间尺度**：超状态上的 actor（策略）更新。
  3.  **最慢时间尺度**：树状抽象结构本身的更新（叶节点的展开与合并）。

### 关键技术细节

- **树状抽象表示**：使用四叉树（或其他层次树）表示状态空间，叶节点对应超状态，支持 **Expansion（细化）** 和 **Collapse（粗化）** 操作。
- **Q 值引导的结构更新**：
  - 为每个叶节点维护 Q 值估计，同时维护其**潜在子节点**和**父节点**的 Q 值估计（通过在线采样同步更新）。
  - **扩展条件**：若子节点最佳 Q 值与父节点最佳 Q 值之差大于阈值 \(3(1-\beta)\lambda\)，则展开该叶节点（细化）。
  - **合并条件**：若父节点与子节点的 Q 值差小于阈值，则合并子节点（粗化）。
- **多时间尺度算法流程**：
  1. 初始化树和聚合 actor-critic。
  2. 与环境交互，收集轨迹。
  3. 在超状态空间上，使用两时间尺度 actor-critic 更新值函数和策略。
  4. 在更慢的时间尺度上，根据 Q 值差异对树进行扩展和合并操作（如每若干步执行一次）。
- 软更新机制：展开或合并时，新节点的 actor-critic 参数由旧节点通过加权平均初始化，保持连续性。
