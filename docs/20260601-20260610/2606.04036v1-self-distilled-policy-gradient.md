## 摘要
在策略自蒸馏是一种语言模型利用特权上下文来监督自身生成的方法，对于稀疏奖励强化学习而言，它是一种有前景的密集监督来源。实际上，它可以实例化为一个辅助的全词汇学生到教师的逆Kullback-Leibler散度损失。因此，我们提出了SDPG，一种自蒸馏策略梯度框架，它结合了群体相对验证器优势与归一化标准差、精确的全词汇在策略自蒸馏以及参考策略KL正则化。实验表明，SDPG在稳定性和性能上优于RLVR和自蒸馏基线。代码可在https://github.com/lauyikfung/SDPG获取。

## Abstract
On-policy self-distillation, where a language model conditions on privileged context to supervise its own generations, is a promising source of dense supervision for sparse-reward reinforcement learning. Actually, it can be instantiated as an auxiliary full-vocabulary student-to-teacher reverse Kullback-Leibler divergence loss. We therefore propose SDPG, a self-distilled policy-gradient framework that combines group-relative verifier advantages with normalized standard deviation, exact full-vocabulary on-policy self-distillation, as well as reference-policy KL regularization. Empirically, SDPG improves stability and performance over RLVR and self-distillation baselines. The code is available at https://github.com/lauyikfung/SDPG.

### 1. 论文的核心问题与整体含义（研究动机和背景）

-   **核心问题**：在大语言模型（LLMs）的强化学习训练中，特别是在涉及数学和代码生成等复杂推理任务时，存在一个关键挑战：**稀疏奖励问题**。现有的强化学习方法（如GRPO）通常只在序列结束时提供一个标量的二值奖励（正确/错误），这导致模型难以区分推理过程中具体哪一步是正确的或错误的，从而造成“信用分配”困难，并导致训练效率低下和不稳定。
-   **研究动机**：为了解决稀疏奖励问题，研究者引入了“在线策略自蒸馏”技术。该技术利用模型自身在拥有更丰富上下文（如正确答案或解题思路的“特权信息”）时产生的分布，来监督其在没有特权信息时的标准输出。这可以转化为一个**密集的、逐token的监督信号**，有效缓解了稀疏奖励的痛点。然而，现有方法仍存在一些问题：
    -   **教师模型负担重**：传统知识蒸馏需要一个更大、更强的外部教师模型，增加了计算和内存开销。
    -   **异构信号不稳定**：不同模型产生的教师信号可能不一致，影响训练平滑性。
    -   **纯自蒸馏的局限性**：单纯的自我蒸馏可能会强化“局部合理但全局错误”的轨迹，因为特权教师模型也可能是不完美的。
-   **总体含义**：本文提出一种新的框架SDPG，旨在将**在线策略自蒸馏提供的密集、局部信号**与**强化学习验证器提供的稀疏、全局信号**有机结合，取长补短，从而提升LLM在稀疏奖励场景下的训练稳定性和最终性能。

### 2. 论文提出的方法论：核心思想、关键技术细节

-   **核心思想**：SDPG的核心思想是在一个统一的KL正则化策略优化框架下，融合两个互补的监督信号源：一个来自验证器的稀疏二值信号，另一个来自特权上下文教师模型的密集全词汇蒸馏信号。
-   **关键技术细节与公式**：
    1.  **总目标函数**：SDPG的损失函数由三部分组成：
        `LSDPG(θ) = Lout(θ) + β(k)L+OPD(θ) + α LK(πθ, πref)`
        -   `Lout`: 基于奖励的策略梯度损失，负责提供来自验证器的全局信号。
        -   `L+OPD`: **门控的全词汇在线策略蒸馏损失**，提供密集的逐token蒸馏信号。
        -   `α LK(πθ, πref)`: 对固定参考策略`πref`的KL正则化项，用于稳定策略更新。
    2.  **全词汇在线策略蒸馏（Full-Vocabulary OPD）**：
        -   给定同一个模型，定义学生分布`pt = πθ(a|x, y<t)`（无特权上下文）和教师分布`qt = πθ(a|c, x, y<t)`（有特权上下文）。
        -   损失函数为**学生到教师的逆KL散度**`DKL(pt || SG[qt])`，其中SG表示停止梯度算子（教师分支不更新）。
        -   该损失函数在计算时是**全词汇**的，即在每个时间步计算整个词汇表上的KL散度，而不是仅采样一个token，这提供了更精确的梯度估计。
        -   **理论贡献**：论文证明（Proposition 3.1），对于固定的前缀，该逆KL损失对学生侧的梯度等价于一个基于采样的策略梯度形式，其token优势`Adist_t(a)`是**中心化的对数教师/学生比率**。这解释了自蒸馏如何像一个策略梯度一样工作。
    3.  **稳定策略**：
        -   **正优势门控**：只有当某个响应的组内优势`Aout(i) > 0`（即验证器认为该响应比组内平均水平好）时，才计算该响应上的蒸馏损失`L+OPD`。这防止在“全局错误”的轨迹上进行无用的自蒸馏，避免与验证器信号冲突。
        -   **β调度器**：蒸馏系数`β(k)`遵循一个先预热（warmup）再退火（decay）的规划。早期预热防止初始不准确的教师信号干扰；后期退火是为了避免在训练后期过度约束学生，鼓励其进一步探索。
    4.  **非归一化KL正则化**：对于参考策略正则化项，论文采用了非归一化KL（UKL）变体，相比标准KL，UKL在梯度计算上更简洁、对称，并能避免一些估计偏差。
